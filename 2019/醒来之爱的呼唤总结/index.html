醒来之爱的呼唤总结

发布 : 2019-04-03 分类 : 笔记 浏览 :

尊敬的温州菩提子影视传媒有限公司!

尊敬的北京大爱菩提民间公益团队!

尊敬的佟老师爱心团队!群里的义工朋友老师们!

尊敬的在为电影（醒来之爱的呼唤）中参与制作，出演，推广，传播助力过程中，积极奔走的义工老师朋友们!
您们的大爱体现在一场场的点映汇报中!

祝大家六时吉祥!爱心满满，能力能量提升的同时，让世界因您们而美丽，让我们因您们的参与而骄傲和自豪!

2019年的妇女节前夜，一份大礼从天而降，就是<醒来之爱的呼唤>电影播映的信息！在不知道内容的时候凭直觉看标题就知道一定是值得推广和观看的。在参与点映的过程中，和天津点映的发起人商美玲老师有缘相遇，非常感恩这次的相遇让我从商美玲那里学到了真诚、务实、虚心和敬业!

由于前期没有及时和老师沟通，以为出钱安排家人观影就可以了，所以价位谈的低目的是多播映几场。后来从高小玲那里知道了电影推广的运作出票，上报等!和影院多次的洽谈也知道了剧院的运作模式和我们打埋伏捉迷藏。为了便于后面师兄的包场，了解整个运作流程我们建立了天津市助力<醒来……>义工群。义工群初期运作时每天都在协调，磨合，十一点多了还在商议，义工们自发的组织护持提建议出点子，说干就干，没有一点耽搁，积极性高涨，一心一意全心全意，只为<醒来……>，事实证明同心同德步调一致才能走的久!

比如姜秋玲老师中午看到义工服缺少的信息以后，马上开车去取义工服回家以后洗义工服然后嗮了以后看到材料需要提取以后，不停歇的开车取材料从中午到晚上六点半点映没有一刻的歇息。

比如熙悦汇那场，商老师听说宣传材料没有落实以后大力支持从杨柳青会场协调，播映以后送到南开播映以后在带回去。还有冯杰老师也是骑电动车去取布标材料等等感人事迹每天都有!

比如叶晓勤老师孩子临近高考爱人在北京工作，她负责统计爱心善款和票数，打电话核对，条例清晰为后面的运作打下良好基础!

比如张文芳老师为天津市场的推广也是立下汗马功劳!

特别感谢高小玲老师把美好的信息分享与我!

非常感谢天津地区推广人商美玲!

非常感谢佟易诚佟老师，使得我能近距离的接近大德老师!

赠人玫瑰手留余香!2019年因<醒来……>改变了我们的生命和生活轨迹!

下面是点映现场义工安排：
1、订票联系人一名便于了解订票数量
2、领票和发票的人员二名
3、宣传材料发放专人负责
4、现场摄影留影人员一名
5、全场监督指导一人
6、义工团队引领十人由专人负责
7、播映以后专人负责现场布标材料的收放和下一包场负责人的衔接。

从第一场开始，领队把模式传递给下一个爱心人士，细节包括联系影院，出票，开票，宣传材料，布标的安排等事宜!

观影出票运作过程中发生许多的问题，及时处理和解决!负责人之间随时联系。如果观影订票人数多可以一场一场的安排也可以改为大的放映厅。比如高小玲助力团队的包场原计划120人因为报名人数太多增加到240人。观影的学生家长看后深受启发和感动，发心包场，接力的义工询问发心者的冠名是个人还是单位，如果都没有要求会征求发心者冠名天津雨花斋助力团队可以以后，冠名天津雨花斋助力团队。

各个地方的助力团队如果有义工服最好，义工服提供可以从义工团队借用。

一个城市的电影排序按时间顺序最好有统一格式，便于后期接龙，这样时间场次地点联系人一目了然。

因为前期的运作经过推敲调整安排基本定型，所以后面复制就可以了，一场一场的循环就是这样开始。

爱心人士包场过程中遇到问题，上位师兄一定大力支持和协调安排，把模式传递下去。

一个地方最好建立义工群，在群里把遇到的问题交代清楚
订票联系人一名
取票发票二人
引领义工最少十人
发放宣传材料二到四人
助力义工听从领队安排
做好各地区影院的协调！
最后非常感恩提供优质影片的制作团队及其推广过程中助力的朋友老师们!

“自新归来的父亲，叛逆不羁的儿子，痛苦绝望的母亲。”一部让爱回归、让灵魂觉醒的震撼心灵之作，使志愿者们在潸然泪下之中感受着中华优秀传统文化的魅力。
