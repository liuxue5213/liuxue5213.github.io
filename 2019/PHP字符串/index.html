PHP字符串

发布 : 2019-04-21 分类 : PHP 浏览 :

#URL

http_build_query($array,[string $numeric_prefix,[string $arg_separator,[int $enc_type=PHP_QUERY_RFC1738 ]]]) 生成URL-encode之后的请求字符串

numeric_prefix 前缀
arg_separator 分隔参数
enc_type 默认使用PHP_QUERY_RFC1738
PHP_QUERY_RFC1738：urlencoded媒体类型进行编码，空格会被编码成加号（+）
PHP_QUERY_RFC3986：空格会被百分号编码（%20）

get_hearder($url,[int $format = 0 ]) 取得服务器响应一个 HTTP 请求所发送的所有标头

$format：如果参数设为1，则get_headers()会解析相应的信息并设定数组的键名
[Content-Length] => 438
[Content-Type] => text/html

get_meta_tags($filename，[bool $use_include_path=false]) 从一个文件中提取所有的meta标签content属性，返回一个数组

$filename：HTML文件的路径字符串。此参数可以是本地文件也可以是一个URL
$use_include_path:设置为TRU 将使PHP尝试按照include_path标准包含路径中的每个指向去打开文件。这只用于本地文件，不适用于URL

base64_encode() 设计此种编码是为了使二进制数据可以通过非纯 8-bit 的传输层传输

Base64-encoded 数据要比原始数据多占用33%左右的空间

base64_decode()

urlencode() 此函数便于将字符串编码并将其用于URL的请求部分，同时它还便于将变量传递给下一页

空格=>+ -_不变 其余% 16进制

urldecode()

rawurlencode() 按照RFC1738对URL进行编码

rawurldecode()

explode(‘’,$arr) 拆分

implode(‘’,$arr) john 合并

parse_str($str,[$arr]) 将字符串解析成多个变量

URL传递入的查询字符串（query string），则将它解析为变量并设置到当前作用域
当前的QUERY_STRING:可以使用$_SERVER[‘QUERY_STRING’]
$arr:变量将会以数组元素的形式存入到这个数组，作为替代

parse_url($url,[$component = -1]) 解析URL，返回其组成部分

$url:要解析的URL,无效字符将使用 _ 来替换
$url = ‘http://username:password@hostname/path?arg=value#anchor';
print_r(parse_url($url));
echo parse_url($url, PHP_URL_PATH);

#编码

ord($string) 返回字符串的ascii值 单字符

chr($ascii) 返回指定的字符

convert_uuencode($str) 使用 uuencode 编码一个字符串

convert_uudecode($str) 解码一个 uuencode 编码的字符串

#加密解密

crypt($str,[$salt]) 单向字符串散列

md5($str，[$raw_output=false]) 计算字符串的 MD5 散列值

md5_file($filename，[$raw_output=false]) 计算文件散列值

true 则返回16进制格式

sha1($str，[$raw_output = false]) 计算字符串的sha1散列值

TRUE，将以20字符长度的原始格式返回
否则返回值是一个40字符长度的十六进制数字

sha1_file($str，[$raw_output = false]) 计算文件的 sha1 散列值

#大小写

ucword($str) 每个单词首字符大写

ucfirst($str) 使第一个字符大写

lcfirst($str) 使第一个字符小写

strtoupper() 将字符串转化为大写

strtolower() 将字符串转化为小写

metaphone($str，[int $phonemes = 0]) 为发音接近的单词 创建相同的键

metaphone(‘programming’);
metaphone(‘programmer’);
“PRKRMNK” “PRKRMR”

similar_text($first,$second，[float &$percent]) 计算两个字符的相似度

$percent:通过引用方式传递第三个参数，similar_text()将计算相似程度百分数。

str_replace($search,$replace,$str,[$count]) 替换

$count:被替换的次数

str_ireplace($search,$replace,$str,[$count]) 忽略大小写版本

str_repeat($input，$multipiler) 重复字符串多少次

str_pad($str,$pad_length,[$pad_string,[$pad_type = STR_PAD_RIGHT ]] ]]) 使用另一个字符串填充到指定长度

从左端、右端或者同时两端被填充到制定长度后的结果
pad_length的值是负数，小于或者等于输入字符串的长度，不会发生任何填充

str_rot13($str) rot13转换

ROT13编码简单地使用字母表中后面第13个字母替换当前字母，同时忽略非字母表中的字符
编码和解码都使用相同的函数，传递一个编码过的字符串作为参数，将得到原始字符串

//构造字符串
$str = “ABCDEFGHIJKLMNOPQRSTUVWXYZ”;
echo “原字符串：”.$str.””;

//按各种方式进行截取
$str1 = substr($str,5);
echo “从第5个字符开始取至最后：”.$str1.””;

$str2 = substr($str,9,4);
echo “从第9个字符开始取4个字符：”.$str2.””;

$str3 = substr($str,-5);
echo “取倒数5个字符：”.$str3.””;

$str4 = substr($str,-8,4);
echo “从倒数第8个字符开始向后取4个字符：”.$str4.””;

$str5 = substr($str,-8,-2);
echo “从倒数第8个字符开始取到倒数第2个字符为止：”.$str5.””;
?>

支持中英文混合截取。
$str_cut 需要截断的字符串
$length 允许字符串显示的最大长度

程序功能：截取全角和半角（汉字和英文）混合的字符串以避免乱码
