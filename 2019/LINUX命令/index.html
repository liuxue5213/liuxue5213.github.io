Linux命令

发布 : 2019-07-18 分类 : LINUX 浏览 :

当前时间 date
默认显示当前月的日历 cal
磁盘剩余空间 df
内存使用情况 free

更改目录到之前的工作目录 cd -

查看文件类型 file 1.txt
查看目录下所有文件 ls -a
查看多个目录 ls /home /usr/bin
查看目录长格式 ls -l
结果按照大小排序 ls -S
结果按照修改时间排序 ls -t

-rw-r–r– 1 root root 453764 2007-04-03 11:05 oo-welcome.odt
权限 -普通文件 d目录 文件所有者权限 所属组成员的权限 其他人的权限
文件硬链接数量 文件的用户名 文件所属的用户组名
文件大小 上次修改时间 文件名

查看文件 上下滚动方式 less

系统目录

bin 系统启动、二进制文件
boot 内核、磁盘镜像、启动加载程序
dev 设备节点的目录
etc 系统层面的配置文件
home 系统分配给不同用户的目录
lib 核心系统程序所需的库文件
media 可以移除媒体设备的挂载点
mnt 可以移除设备的挂载点
opt 安装可选软件的存储目录
proc 内核维护的虚拟文件系统
root root用户的主目录
sbin 系统的二进制文件
tmp 程序创建的临时文件、重启会被删除
usr 普通用户所需要的所有软件和文件
usr/bin 系统安装的可执行程序
usr/lib usr/bin程序的共享库
usr/local 编译安装的程序存储目录
usr/sbin 系统管理程序
usr/share usr/bin程序的共享数据
usr/share/doc 安装在系统中软件的文档
var
var/log 日志文件、系统活动的记录

创建文件夹 mkdir
复制文件或目录 cp
移动/重命名文件或目录 mv
删除文件或目录 rm
创建硬链接和符号链接 ln -s item link

解释一个命令 type
显示程序的执行目录 which
命令手册 man
显示一系列适合的命令 apropos
显示命令的简洁描述 whatis
创建别名 alias name = ‘’

cat
排序文本行 sort
uniq
grep
wc 输出文件的行数、单词数、字节数
输出文件的第一部分 head -n 100 1.txt
输出文件最后的一部分 tail -f 100 1.txt
内容加入到文件中 111111 > 1.txt
历史命令 history !序号 直接使用

/dev/null 位存储桶 接收输入 不做任何处理
echo {a..z} a b c d e …
mkdir {2007..2009}-0{1..9} {2007..2009}-{10..12}

移动光标

ctrl a 行首
ctrl e 行尾
ctrl f 前移一个字符
ctrl b 后移一个字符

ctrl l 清空屏幕
ctrl a 行首
ctrl d 删除光标位置的字符
ctrl t 光标位置的字符和前面的字符互换位置

ctrl k 剪切从光标到行尾的文本
ctrl u 剪切从光标到行首的文本

alt f 前移一个字
alt b 后移一个字
alt l 光标位置到字尾的字符转换为小写
alt u 光标位置到字尾的字符转换为大写

系统

id 用户身份号
umask 设置默认的文件权限
su 以另一个用户的身份来运行shell
sudo 以另一个用户的身份来执行命令
chmod 更改文件模式
chown 更改文件所有者
chgrp 更改文件组所有权
printenv 打印环境变量
export 导出环境变量

进程

ps 当前进程快照
top 显示任务
kill 给一个进程发送信号
killall 关闭进程

jobs 列出活跃的任务
bg 一个任务放到后台执行
fg 一个任务放到前台执行

存储

mount 挂载一个文件系统
umount 卸载一个文件系统
fack 检查修复一个系统
fdisk 分区表控制器
mkfs 创建文件系统
dd 写入设备
md5sum 计算MD5检验码

包管理

debian dpkg apt-get,aptitude
fedora,red hat,centos rpm yum

查找
debian apt-get udpate dpkg –install name
apt-cache search name
red hat yum search name rpm -i name

确认是否安装
debian dpkg –status name
red hat rpm -q name
