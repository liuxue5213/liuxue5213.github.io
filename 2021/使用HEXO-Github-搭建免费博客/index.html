使用HEXO Github 搭建免费博客

一、 安装环境

  1. 安装Node https://nodejs.org/en/
  2. 安装Git windows https://git-scm.com/download/
    mac os 基本上都是自带 brew install git
    linux sudo apt-get install git
  3. 申请GitHub https://github.com/ 当做远程仓库、域名、服务器，托管博客的静态页面。

二、 安装Hexo

  1. 创建一个文件夹,如hexo,作为存放hexo的文件,然后进入hexo文件夹里安装Hexo。
  2. 使用npm安装Hexo：sudo npm install hexo-cli -g
  3. 执行init初始化hexo: hexo init
    (如果init失败,也可以手动生成文件 hexo install )，全部安装完成,blog为根目录,所有的操作都在里面进行。

三、 生成静态页面

所有的文章在 source/_posts, 文章支持Markdown语法，可以使用一些MarkDown渲染工具。

启动本地服务，进行文章预览调试，默认端口4000，按下ctrl + c关闭server

浏览器输入http://localhost:4000 就可以看到生成的静态博客了

建立Repository https://github.com/用户名
建立与用户名对应的仓库，仓库名必须为【用户名.github.io】，此为仓库的固定写法
