Linux基礎

一、 簡介

Linux 是一套免费使用和自由传播的类 Unix 操作系统，是一个基於 POSIX 和 Unix 的多用户，多任务，支持多线程和多 CPU 的操作系统。它能运行主要的 Unix 工具软件，应用程序和网络协议。它支持32 位和 64 位硬件。Linux 继承了 Unix 以网络为核心的设计思想，是一个性能稳定的多用户网络操作系统。

Linux 严格来说是单指操作系统的内核，因操作系统中包含了许多用户图形接口和其他实用工具。如今 Linux 常用来指基於 Linux 的完整操作系统，内核则改以 Linux 内核称之。由於这些支持用户空间的系统工具和库主要由理查德·斯托曼在 1983 年发起的 GNU 计划提供，自由软件基金会提议将其组合系统命名为 GNU/Linux，但 Linux 不属於 GNU 计划，这个名称没有得到社区的一致认同。

Linux 最初是作为支持英特尔 x86 架构的个人电脑的一个自由操作系统。目前 Linux 已经被移植到更多的计算机硬件平台，进进超出其他任何操作系统。Linux 可以运行在服务器和其他大型平台之上，如大型计算机和超级计算机。世界上 500 个最快的超级计算机已 100％运行 Linux 发行版或变种。Linux 也广泛应用在嵌入式系统上，如手机（Mobile Phone）、平板电脑（Tablet）、路由器（Router）、电视（TV）和电子游戏机等。在秱劢设备上广泛使用的 Android 操作系统就是创建在 Linux 内核之上

通常情况下，Linux 被打包成供个人计算机和服务器使用的 Linux 发行版，一些流行的主流 Linux发布版，包括 Debian（及其派生版本 Ubuntu、Linux Mint）、Fedora（及其相关版本 Red Hat Enterprise Linux、CentOS）和 openSUSE 等。Linux 发行版包含 Linux 内核和支撑内核的实用程序和库，通常还带有大量可以满足各类需求的应用程序。个人计算机使用的 Linux 发行版通常包含 X Window 和一个相应的桌面环境，如 GNOME 或 KDE。桌面 Linux 操作系统常用的应用程序，包括 Firefox 网页浏览器、LibreOffice 办公软件、GIMP 图像处理工具等。由於 Linux 是自由软件，任何人都可以创建一个符合自己需求的 Linux 发行版

二、 Linux 优点

Linux 由众多内核组成，其源代码完全开源；

Linux 继承了 Unix 的特点，具有非常强大的网络功能，其支持所有的因特网协议，包括 TCP/IPv6和链路层拓扑程序等，且可以利用 Unix 的网络特性开发出新的协议栈；

Linux 系统工具链完整，简单操作就可以配置出合适的开发环境，可以简化开发过程，减少开发
中仿真工具的障碍，使系统具有较强的秱植性

三、 常用命令

1. 目錄操作

pwd 显示当前目录
cd 改变当前目录
ls 显示目录中的文件和子目录
-h 易读的方式显示文件大小
-a 显示隱藏文件
-l 目录的文件大小 自動顯示
mkdir 创建目录 mkdir
-p a/b/c 自動创建父目录
cp 将文件复制到另一个位置
-p 保留文件权限，拥有者，时间戳属性
mv 将文件移動到另一个位置（可用於重命名）
rm 删除一个文件
-f 强制删除文件，不提示确认信息
-r 删除所有文件和子目录
chmod 改变文件权限
4：读（r）
2: 写（w）
1: 执行（x）
chmod 755 1.txt
chmod +x 1.txt

2. 测试命令

ping 网络連通性测试工具
-c N 发送 N 个数据包

wget 一个非交互的网络下载工具
-b 后台下载
-O path 保存到指定目录

3. 進程管理

ps 显示在本地机器上当前运行的進程列表
ps aux 查看所有進程
状态说明（STAT） D 表示不可中断的睡眠状态
R 表示运行或就绪状态
S 表示可中断的睡眠状态
T 表示停止状态
Z 表示僵尸状态
< 表示高优先级進程
N 表示低优先级進程

  • 表示前台迚程组中的進程
    l 表示多线程進程
    s 表示会话進程的首進程

排序：
ps -aux –sort -pcpu | more 按 cpu 占用百分比做降序排序
ps -aux –sort -rss | more 按内存占用百分比做降序排序

top 命令：交互查看進程和系统相关属性信息，默认为 3 秒刷新一次

前五行是系统的整体的统计信息
top - 21:06:13 up 7:40, 2 users, load average: 0.00, 0.01, 0.05
任务: 190 total 1 running, 189 sleeping, 0 stopped, 0 zombie
%Cpu(s): 0.3 us, 0.2 sy, 0.0 ni, 99.5 id, 0.0 wa, 0.0 hi, 0.0 si, 0.0 st
MiB Mem : 1951.6 total, 1064.3 free, 483.9 used, 403.3 buff/cache
MiB Swap: 975.0 total, 975.0 free, 0.0 used. 1319.3 avail Mem

任务队列信息(第一行)
top - 21:06:13 当前时间
up 7:43 系统运行时间，格式为时:分
2 users 当前登录用户数
系统负载，即任务队列的平均长度。 三个数值分别为 1 分钟、5分钟、15 分钟前到现在的平均值。 load average: 0.00, 0.01, 0.05

進程和 CPU 的信息( 第二、三行)
当有多个 CPU 时，这些内容可能会超过两行。按 1 键时可以切换显示 CPU 的信息
任务: 190 total 進程总数
1 running 正在运行的進程数，如果是一个 CPU 一般只有一个，如果有两个 CPU 则会有 2 个
189 sleeping 睡眠的進程数
0 stopped 停止的進程数
0 zombie 僵尸進程数

%Cpu(s): 0.3 us 系统用户進程使用 CPU 百分比
0.2 sy 内核中的進程占用 CPU 百分比
0.0 ni 用户迚程空间内改变过优先级的進程占用 CPU 百分比
99.5 id 空闲 CPU 百分比
0.0 wa 等待 IO 完成所占用的时间百分比
0.0 hi 硬件中断所占用的时间百分比
0.0 si 软件中断所占用的时间百分比
0.0 st 被偷走的时间百分比

内存信息(第四五行)

MiB Mem : 1951.6 total 物理内存总量
1064.3 free 空闲内存总量
483.9 used 使用的物理内存总量
403.3 buff/cache 用作内核缓存的内存量。和 free -k 一个意思

MiB Swap: 975.0 total 交换区总量
975.0 free 空闲交换区总量
0.0 used 使用的交换区总量
1319.3 avail Mem 总的可利用内存是多少

進程信息

PID 進程id
PR 优先级
USER 進程所有者的用户名
NI 進程优先级。 nice 值。负值表示高优先级，正值表示低优先级
VIRT 虚拟内存集
RES 实际使用内存大小，长驻内存集
SHR 共享内存大小
S 進程状态
D=不可中断的睡眠状态
R=运行
S=睡眠
T=跟踪/停止
Z=僵尸進程
%CPU 上次更新到现在的 CPU 时间占用百分比
%MEM 進程使用的物理内存百分比
TIME+ 進程使用的 CPU 时间总计，单位 1/100 秒
COMMAND 命令名/命令行

TOP 的快捷键

默认 3s 刷新一次，根据 CPU 占用百分比排序
空格：立即刷新
q：退出
M：按内存排序
P：按 CPU 排序
T：根据在 CPU 中运行的累计时间進行排序
l：是否显示平均负载和启劢时间
t：是否显示運行和 CPU 状态相关信息
m：是否显示内存相关信息
c：是否显示完整的命令行信息
k：终止某个進程
d：更改刷新时间

kill 给進程发送一个信号
kill -9 1024 关闭 PID 为 1024 的迚程
service 管理系统中的服务
service apache2 status 查看服务状态
service apache2 stop|start 停止|启劢服务
netstat 显示网络统计信息，TCP/IP 网络連接
-a 显示所有违接
-n 禁止解析 IP 地址和端口号
-t 显示 TCP 违接

ifconfig 管理网络适配器
ifconfig eth0 192.168.1.100 netmask 255.255.255.0 给 eth0 配置 IP 地址和子网掩码

4. 用户管理

useradd 创建，更新用户
useradd test 创建用户
-s 指定登录 shell
passwd 修改用户密码
passwd 123456
userdel 删除一个用户不其相关的文件
userdel -r test 删除 test 用户，併删除目录和邮件
su 更改用户 ID 或成为超级用户
su root 切换到 root 用户
sudo 以超级用户的身份执行命令
sudo ifconfig 以 root 身份执行命令

5. 信息查詢

whoami 显示当前登录账户
history 查看 shell 歷史命令
-c 清除歷史记录
uname 输出系统信息
-a 显示所有信息
-r 显示内核的发行版本号
shutdown 关闭，重启计算机
-h 关闭计算机
-r 重新启劢计算机
shutdown -h 0(now) 立刻关闭计算机

6. VIM编辑器

三种模式：
命令模式：一键啟動 vim,就進入命令模式
输入模式：按 i 進入输入模式
底行模式：按：（冒号）進入底行模式

命令模式：
gg 跳到第一行
G 跳到最后一行
dd 剪切当前行
p 粘贴到下一行
yy 复制当前行
dG 從当前删除到文件结尾
/keyword 关键词搜索

输入模式：按 i 進入输入模式，开始输入文件内容

底行模式：
:w 保存内容
:q 退出 vim
:q! 强制退出 vim,不保存修改内容
:3 跳到第三行
