4信息搜集

发布 : 2021-03-29 分类 : 网络安全 浏览 :

网络安全(一)

一、 信息搜集

1、 信息搜集要获取的内容

社工层面：
公司地址、邮件地址、域名信息、联系电话、公司职务人员

技术层面：
IP 地址、端口信息、操作系统类型、技术架构、服务版本信息、敏感信息、子域名

總結：
网站服务器方面：服务器系统及版本，端口信息，IP 地址，IP 地址牵扯到 CDN 防护；
站点信息：中间件、编程語言、数据库、后台目彔、敏感目彔、敏感文件、旁站、C 段；
子域名信息收集：注册信息、备案信息、子域名信息收集
网站维护人员信息：姓名、职位、邮件信息、联系方式
公司信息：公司地址、公司法人、公司性质、公司服务等

查询网站的域名和本案信息

whois www.baidu.com
阿里云域名查询
企查查

解析ip地址
https://www.wepcc.com/
https://tools.ipip.net/ping.php

2、 主动信息搜集

与目标主机直接进行交互，从而获取到目标信息，缺点会产生操作记录

例如：nmap端口扫描 scapy高级扫描

解决方法：
使用第三方设备、主机检测；
连接代理，隐藏ip；
在网络模型2、3、4层，进行数据探测；

3、 被动信息搜集

不与目标主机直接进行交互，使用第三方或者其他方式 获取目标信息；

Google网上搜索； shadan；fofa；主动 nmap扫描;

二、知识扩展

1、 常用端口

21 ftp
22 ssh
23 telnet 终端仿真协议 未加密文本通信
25 smtp 简单邮件传输协议
80 超文本传输协议 用于网页

443 超文本传输安全协议 tls/ssl加密传输
445 smb 文件共享
3306 mysql 数据库
3389 远程桌面协议 rdp

2、 域名

3级子域名.2级域名.顶级域名

全球有13个根域

A 正向解析 域名和ip
Cname 别名记录 多个域名解析到同一个ip
MX 邮件服务器 邮箱后缀
PTR 反向解析 ip => 域名

dns 缓存域名解析结果

浏览器 从本地 缓存 查询dns信息
查询根服务器 反馈结果
从顶级域名服务器查询 返回主服务器
从主服务器 查询ip地址 返回
本地缓存

5、 cdn

CDN的基本原理是广泛采用各种缓存服务器，将这些缓存服务器分布到用户访问相对集中的地区或网络中，在用户访问网站时，利用全局负载技术将用户的访问指向距离最近的工作正常的缓存服务器上，由缓存服务器直接响应用户请求
