行测-资料分析

发布 : 2020-10-10 分类 : 公考 浏览 :

行测学习/实战策略

核心战略：集中优势火力，猛烈攻击有效得分区域！

有效：难度小，容易得分的，单体分值小的或者分值高的，也就是言语理解与表达，数量关系，资料分析；

1、题目难度：常识＞数量＞言语＞判断＞资料

常识 抢时间，将更多的时间合理运用；
每分钟3题；正式考试前 就要开始看
数量 抢分数，拉开差距的模块

言语
判断 基础题
资料 占20分，并且考试难度相较于其他模块，偏小；
推荐时间25-30分钟左右

2、答题目标：快速准确地选出答案

选出答案：能估算 就不计算

3、答题顺序：

①找到适合自己的答题顺序

适合自己的答题：做5套题以上，哪个模块正确率高，哪个模块就是最擅长的，然后优先做这个模块

②大原则：需要冷静分析、推理的先做

优先资料分析，逻辑推理；
图推，类比推理可放在后面；

4、涂卡顺序：做一个模块，涂一个模块
涂完需要5-8分钟，不要从1-135；
如果涂错1道题，后面都会错；

5、学会放弃：1分钟，没有思路坚决放弃（30秒原则）
正常情况下，最多2分钟；没有思路跳过进入下一题

资料分析学习/实战策略

1、辅助工具：直尺、量角器、手表
直尺 画横线、量刻度、撕卷子
量角器 量饼图
手表 钟表问题，一半 一半

2、圈重点：时间、主语、重点字

时间 现期、基期
主语 文字材料的主语
重点字 定题型

3、计算方法：割补法、截位法、三位有效数字直除……（估算法）

4、题型及做题顺序：图形→表格→综合→文字(数据不好找)

5、数字敏感度：

资料分析高分必做

1.公式和百化分背到烂熟！
2.整理所有知识点的出题标志！
3.整理错题、经典例题、高频考点！
4.每道题都必须圈重点！圈重点！圈重点！
5.用基础讲义了解估算法，用套题熟练估算法！
6.改变硬算的做题习惯，重点学习处理数据的技巧！

1、 出题标志
2、 公式
3、 注意事项
4、 总结典型例题3-5题以上

统计术语

基期、现期

基期 基础时期、时间靠前
现期 现在时期、时间靠后

1、2016年的销售额是1000万元，2017年销售额为1200万元

基期 2016年1000
现期 2017年1200
增长了 1200 - 1000 = 200

2、 2020年销售额是1000万元，增长率为20%，如果明年保持同样的增长速度，2021年的销售额将达到多少

现期 = 基期 * (1 + 增长率) = 1000 * (1+20%) = 1000 * 1.2 = 1200

同比：与历史同期相比较
注释：同比实际是指与去年的同一时期相比较
年份减一

环比：现在统计周期和上一个统计周期相比较，包括周环比、月环比、年环比
注释：环比实际上是指“与紧紧相邻的统计周期相比较”
最小的时间周期减一
