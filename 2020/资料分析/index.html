资料分析概述

资料分析主要是测查报考者对各种形式的文字、图表等资料的综合理解与分析加工的能力，这部分内容通常由统计性的图表、数字及文字材料构成

针对一段资料一般有1-5个问题，报考者需要根据资料所提供的信息进行分析、比较、推测、计算，从四个备选答案中选出符合题意的答案

备选答案：
1、不看选项，不计算(数量+资料)
2、选项也是已知条件

先看重点字，定题型；
再看主语；
带主语找数据，代入公式

资料分析解题步骤及阅读技巧练习

做题顺序： 1 ->-> 2 ->-> 3
1、 材料 时间，主语(+关系：总分、并列)
2、 问题 时间，主语，重点字 => 出题标志
3、 选项 把算式或数据，抄到题目旁边 或 选项后面

资料分析之“结构阅读法”

通用重点阅读对象：材料标题、时间表述、单位表述、主语、注释
文字类 30秒
表格类 15秒
图形类 10秒

参考时间：不超过1分钟
计算时，看单位，每一段最多3个主语；超过3段以上，每一段大概率一个主语

第五题快速解题
1、先看加减法的选项
2、先看现期数据的选项
3、先看持续/趋势类的选项
4、先看绝对值(最值)的选项
5、先看两期比重变化(类)的选项
6、先看所有简单到一看就出来结果的选项
7、先看主语一致的选项(进出口问题)
8、跳过项数太多的选项，项数少的优先计算

统计术语

基期、现期

基期 基础时期、时间靠前
现期/估算将来时 现在时期、时间靠后

1、2016年的销售额是1000万元，2017年销售额为1200万元

基期 2016年1000
现期 2017年1200
增长了 1200 - 1000 = 200

2、 2020年销售额是1000万元，增长率为20%，如果明年保持同样的增长速度，2021年的销售额将达到多少

现期 = 基期 * (1 + 增长率) = 1000 * (1+20%) = 1000 * 1.2 = 1200

同比：与历史同期相比较
注释：同比实际是指与去年的同一时期相比较
年份减一

环比：现在统计周期和上一个统计周期相比较，包括周环比、月环比、年环比
注释：环比实际上是指“与紧紧相邻的统计周期相比较”
最小的时间周期减一

2018年 2018年上半年 2018年第一季度 2018年11月
同比 2017年 2017年上半年 2017年第一季度 2017年11月
环比 2017年 2017年下半年 2017年第四季度 2018年10月

基期量、现期量、增长率

基期量 = 现期量 - 增长量 = 现期量 / (1 ± 增长率)

现期量 = 基期量 + 增长量 = 基期量 * (1 + 增长率)

口诀：增长为加，下降为减，今年作乘，去年作除

基期量出题标志：基期时间 + 主语 (量的单位)

现期量出题标志：如果明年保持同样的增速

化除为乘法

使用题型：计算基期量（范围：|a%| <= 10%）

a% = 1 / N a% * N = 100

例：2017年的产量是678.12吨，较2016年下降3.27%
问2016年的产量是多少？

基期量 = 现 / (1 - 增长率)
= 680 / (1 - 3%)
= 680 * (1 + 3%)
= 678 +20 = 698

3位有效数字直除(95%+) P156

前提：选项的位数相同

1、 如果选项的首位不同，用三位数有效数字直除，求出首位即可
2、 如果选项的首位相同，第二位不同，用三位有效数字直除，求出第一位，看出第二位

大原则：保留的位数，比选项区别出大小的位数多一位

A.25506 B.172220 C.147942 D.153679

134249 / (1 + 10.2%)

错位相加：一个数乘以 1+10% 或者 1+1%

增速最快 => 增长率最大

增长率

增长率 = (增长量 / 基期量) * 100%
= (现期量 - 基期量) / 基期量
= 增长量 / (现期量 - 增长量)
= (现期量 / 基期量) - 1

2、 比较增长率，增速最快/增幅最大/增加比例最大/变化幅度最大，|a%|最大

减小误差

减少误差，分子分母同升同降，
尽量估算第三位及第三位以后，
估算量占数据的百分之多少

涨跌幅度

增长率 8% -5%

倍数常见的出题题型(或常用语第五题)

1、 是 N 倍 = 多 N-1 倍
2、 约是N倍，N倍上下都符合
3、 N倍多，必须满足大于 N 倍，小于 N+1 倍；
两成多，20-30之间，20% < a% < 30%
4、 超过 N 倍，所有大于 N 倍的都符合
5、 接近 N 倍，略小于 N 倍
6、 扩大 N 倍 = 多 N 倍 = 增长了 N 倍 = 是 N+1 倍

比较大小 需要列算式，抄数据，只有估算的数据

结论：连续时间区间中，基期量差别不大

如果比较增长率，重点比较增长量

增长量

增长量 = 现期量 - 基期量
= 基期量 * 增长率
= (现期量 * 增长率) / (1 ± 增长率)

环比量 = 本月 - 上月

公式一：增长量 = (现期量 * a%) / (1 ± a%)

比较增长量，只用分子来估算

公式二：增长量 = 现 / (N + 1) 减少量 = 现 / (N - 1)

N一般为整数，也有小数

注意：
a.只有选项差距很小，且百化分不精准，才用公式一计算增长量(比例较小，很少会计算)
b.如果比较大小时，选项很接近，用公式二来计算 1300 * 25% 1500 * 22%
c.增长率远超100%时，用公式二来比大小 2000 3000% 2000/0.33 2000 * 300%

比重、百分数、百分点

比重 = 部分 / 总量
总量 = 部分 / 比重
部分 = 总量 * 比重
利润率 = 利润 / 营业收入
资产负载率 = 负债额 / 资产总额
人口增长率 = 出生率 - 死亡率

比重：总体中某部分 占 总体的百分比

部分 => 总体 包含关系 比谁减谁，比谁除以谁

百分数：量A占量B的百分比例： (A / B) * 100% 常见考点：增长率

百分点：两个百分数做差的结果 a% - b% = c%

1、 量与量比较，只能用百分数

例：甲占乙的X% 甲 / 乙
甲比乙的X% (甲 - 乙) / 乙
乙比甲的X% (乙 - 甲) / 甲 绝对值
今年比去年增加/减少X% (现期 - 基期) / 基期

2、 率与率比较，一般用百分点
直接做差，后减前(时间上的后和前) 也就是比谁减谁

在同一个总量中，比重可以代替量来计算

比例赋值：把总量赋值为100

顺差、逆差 P35、P46

顺差(出超) = 出口 - 进口
逆差(入超) = 进口 - 出口

对外贸易顺差：在一个时期内，一个国际(或地区)的出口商品额 大于 进口商品额

对外贸易逆差：在一个时期内，一个国际(或地区)的出口商品额 小于 进口商品额

1、 顺差，出 > 进；逆差，进 > 出
2、 净进口额 = 进口额 - 出口额
净出口额 = 出口额 - 进口额
3、 对外贸易总额 = 进出口总额 = 进口额 + 出口额
4、 对外贸易差额 = |进口 - 出口|
5、 进出口总额增长率一定是混合增长率
6、 主题转换：法 => 中 出口：顺差
中 => 法 进口：逆差

GDP(国内生产总值)

指一个国家(地区)所有常住单位在核算期内生产活动的最终结果

指一个国家（或地区）所有国民在一定时期内生产的最终产品和服务价值的总和

CPI(居民消费价格指数)

反映居民家庭购买消费品及服务价格水平变动情况的相对数（指数的基期数值定为一百）

它是进行经济分析和决策、价格总水平监测和调控及国民经济核算的重要指标

其按年度计算的居民消费价格指数变动率通常被用来作为反映通货膨胀或紧缩程度的指标

基尼系数

定量测定收入分配差异程度

国际上通用的，用以衡量一个国家或地区人民收入差距的常用指标

介于0–1之间，0分配趋向于平等；
越大，代表收入差距越大；

衡量一个国家或地区人民生活水平的常用指标，比例越低，代表生活水平越高

恩格尔系数是食品支出总额占个人消费支出总额的比重

第三年相对于第一年的增长率：R1 + R2 + R1 * R2

隔年倍数公式
第三年是第一年的倍数：R1 + R2 + R1 * R2 + 1

隔年增长率 + 1 = 隔年倍数

如果R1、R2均小于10%，|R1 * R2|远小于1%，可以忽略不计

常考题型
1.计算隔年增长率/隔年倍数
2.计算隔年基期量 = 现期量 / (1 + a%) a%为隔年增长率
3.计算隔年增长量 = 现期量 / (N + 1) a% = 1 / N

隔月增长率，隔多月同样可以计算，不固定

二、混合增长率(混合类…，比重，平均数)

出题标志：量A + 量B = 量C；a% + b% = c%
口诀：
1.居中不正中，偏向量(基期量)大的
2.画图

增长率 = 增长量 / 基期量

比重 = 部分 / 总量

平均数 = 总数 / 总项数

浓度 = 溶质 / 溶液

如果增长率差距不大，则可以使用现期量代替基期量计算

常考题型：
名词类
1.房地产 = 房产 + 地产
2.进出口 = 进口 + 出口
3.全国 = 城市 + 农村
4.总人数 = 男 + 女
5.(夏 + 秋) = 夏粮 + 秋粮
6.总量 = 已知 + 其他
7.水产品 = 水产品捕捞 + 水产品养殖
8.水产品养殖 = 海水养殖 + 淡水养殖
8.水产品捕捞 = 海水捕捞 + 淡水捕捞

时间类
1.全年 = 1-3季度 + 4季度
2.上半年 = 1季度 + 2季度
3.全年 = 1-11月 + 12月
4.上半年 = 1-5月 + 6月

80%以上的考点为混合增长率
1.不同种类，
2.同一种元素，不同的时间区间

三、平均增长率

出题标志：平均/每 + 增长了x%

分子A的增速为a，分母B的增速为b，带入平均数增长率公式 = (a-b) / (1+b)

出题形式：
1.计算平均数增长率，用公式
2.问平均数升降，参照比重变化 画箭头(比较a和b的大小)

平均数增长率 = 现期 - 平均数的基期量 / 平均数的基期量

常考：平均单价、平均产量

平均数增长量(小众考点)
(A / B) - {[A / (1+a)] / [B / (1+b)]}
= (A / B) * [(a-b) / (1+a)]

四、比重的四种常考题型
1.现期比重(默认现期) = 部分 / 整体
出题标志：“占” “中” “比例” “份额” “举足轻重的地位” “比重” “主导地位” “占比50%以上”…

注意：在“第五题”中出现比重，优先判断包含关系

2.现期比重变化
出题标志：“部分” 占 “整体” 的比重高于/低于 上年同期水平
“占…上升/下降…”
计算方法：现期比重 - 基期比重

判断现期比重变化的步骤
a.列式子 分子、分母的主语
b.抄数据 a、b 只抄增长率
c.画箭头 分子越大，上升；分母越大，下降；

3.现期比重变化幅度
出题标志：“部分” 占 “整体” 的比重比上年 上升/下降__个百分点
“占…升降()百分点”
判断现期比重变化的步骤
1.列式子
2.抄数据
3.画箭头
4.选小数

验证是否唯一：是否小于 |a-b| 是选择；否 用公式计算(估算)
箭头方向，在选项中最小的数字，适用90%

(A/B) * [(a-b)/(1+a))]

4.基期比重
出题标志：“基期时间…占…”

估算基期比重的步骤
a.计算A/B
b.抄数据，画箭头

计算现期和升降，估算基期 适用80%题型，然后判断是否唯一

(A/B) * [(1+b)/(1+a))]
