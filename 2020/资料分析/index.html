行测学习/实战策略

核心战略：集中优势火力，猛烈攻击有效得分区域！

有效：难度小，容易得分的，单体分值小的或者分值高的，也就是言语理解与表达，数量关系，资料分析；

1、题目难度：常识＞数量＞言语＞判断＞资料

常识 抢时间，将更多的时间合理运用；
每分钟3题；正式考试前 就要开始看
数量 抢分数，拉开差距的模块

言语
判断 基础题
资料 占20分，并且考试难度相较于其他模块，偏小；
推荐时间25-30分钟左右

2、答题目标：快速准确地选出答案

选出答案：能估算 就不计算

3、答题顺序：

a. 找到适合自己的答题顺序

适合自己的答题：做5套题以上，哪个模块正确率高，哪个模块就是最擅长的，然后优先做这个模块

b. 大原则：需要冷静分析、推理的先做

优先资料分析，逻辑推理；
图推，类比推理可放在后面；

4、涂卡顺序：做一个模块，涂一个模块
涂完需要5-8分钟，不要从1-135；
如果涂错1道题，后面都会错；

5、学会放弃：1分钟，没有思路坚决放弃（30秒原则）
正常情况下，最多2分钟；没有思路跳过进入下一题

资料分析概述

资料分析主要是测查报考者对各种形式的文字、图表等资料的综合理解与分析加工的能力，这部分内容通常由统计性的图表、数字及文字材料构成

针对一段资料一般有1-5个问题，报考者需要根据资料所提供的信息进行分析、比较、推测、计算，从四个备选答案中选出符合题意的答案

备选答案：
1、不看选项，不计算(数量+资料)
2、选项也是已知条件

先看重点字，定题型；
再看主语；
带主语找数据，代入公式

资料分析解题步骤及阅读技巧练习

做题顺序： 1 ->-> 2 ->-> 3
1、 材料 时间，主语(+关系：总分、并列)
2、 问题 时间，主语，重点字 => 出题标志
3、 选项 把算式或数据，抄到题目旁边 或 选项后面

资料分析之“结构阅读法”

通用重点阅读对象：材料标题、时间表述、单位表述、主语、注释
文字类 30秒
表格类 15秒
图形类 10秒

参考时间：不超过1分钟
计算时，看单位，每一段最多3个主语；超过3段以上，每一段大概率一个主语

统计术语

基期、现期

基期 基础时期、时间靠前
现期/估算将来时 现在时期、时间靠后

1、2016年的销售额是1000万元，2017年销售额为1200万元

基期 2016年1000
现期 2017年1200
增长了 1200 - 1000 = 200

2、 2020年销售额是1000万元，增长率为20%，如果明年保持同样的增长速度，2021年的销售额将达到多少

现期 = 基期 * (1 + 增长率) = 1000 * (1+20%) = 1000 * 1.2 = 1200

同比：与历史同期相比较
注释：同比实际是指与去年的同一时期相比较
年份减一

环比：现在统计周期和上一个统计周期相比较，包括周环比、月环比、年环比
注释：环比实际上是指“与紧紧相邻的统计周期相比较”
最小的时间周期减一

2018年 2018年上半年 2018年第一季度 2018年11月
同比 2017年 2017年上半年 2017年第一季度 2017年11月
环比 2017年 2017年下半年 2017年第四季度 2018年10月

基期量、现期量、增长率

基期量 = 现期量 - 增长量 = 现期量 / (1 ± 增长率)

现期量 = 基期量 + 增长量 = 基期量 * (1 + 增长率)

口诀：增长为加，下降为减，今年作乘，去年作除

基期量出题标志：基期时间 + 主语 (量的单位)

现期量出题标志：如果明年保持同样的增速

化除为乘法

使用题型：计算基期量（范围：|a%| <= 10%）

a% = 1 / N a% * N = 100

例：2017年的产量是678.12吨，较2016年下降3.27%
问2016年的产量是多少？

基期量 = 现 / (1 - 增长率)
= 680 / (1 - 3%)
= 680 * (1 + 3%)
= 678 +20 = 698

3位有效数字直除(95%+) P156

前提：选项的位数相同

1、 如果选项的首位不同，用三位数有效数字直除，求出首位即可
2、 如果选项的首位相同，第二位不同，用三位有效数字直除，求出第一位，看出第二位

大原则：保留的位数，比选项区别出大小的位数多一位

例：2016年6月份，城镇消费品零售额23082亿元，同比增长10.5%；乡村消费品零售额3775亿元，同比增长11.2%。1-6月份，城镇消费品零售额134249亿元，同比增长10.2%；乡村消费品零售额21889亿元，同比增长11.0%
按1-6月份的同比增速，2017年1-6月份城镇消费品零售额约为()

A.25506 B.172220 C.147942 D.153679

134249 / (1 + 10.2%)

错位相加：一个数乘以 1+10% 或者 1+1%

增速最快 => 增长率最大

增长率

增长率 = (增长量 / 基期量) * 100%
= (现期量 - 基期量) / 基期量
= 增长量 / (现期量 - 增长量)
= (现期量 / 基期量) - 1

出题标志
1、 计算：增加了/增加/增速/增幅为/增加比例为…/增加了a%/N倍

2、 比较增长率，增速最快/增幅最大/增加比例最大/变化幅度最大，|a%|最大

减小误差

减少误差，分子分母同升同降，
尽量估算第三位及第三位以后，
估算量占数据的百分之多少

涨跌幅度

增长率 8% -5%
增幅(涨幅) 8% -5%
降幅(跌幅) 0 5%
变化幅度(涨跌幅度、取绝对值) 8% 5%

倍数常见的出题题型(或常用语第五题)

1、 是 N 倍 = 多 N-1 倍
2、 约是N倍，N倍上下都符合
3、 N倍多，必须满足大于 N 倍，小于 N+1 倍；
两成多，20-30之间，20% < a% < 30%
4、 超过 N 倍，所有大于 N 倍的都符合
5、 接近 N 倍，略小于 N 倍
6、 扩大 N 倍 = 多 N 倍 = 增长了 N 倍 = 是 N+1 倍

比较大小 需要列算式，抄数据，只有估算的数据

结论：连续时间区间中，基期量差别不大

如果比较增长率，重点比较增长量

增长量

增长量 = 现期量 - 基期量
= 基期量 * 增长率
= (现期量 * 增长率) / (1 ± 增长率)

环比量 = 本月 - 上月

公式一：增长量 = (现期量 * a%) / (1 ± a%)

比较增长量，只用分子来估算

公式二：增长量 = 现 / (N + 1) 减少量 = 现 / (N - 1)

N一般为整数，也有小数

注意：
a.只有选项差距很小，且百化分不精准，才用公式一计算增长量(比例较小，很少会计算)
b.如果比较大小时，选项很接近，用公式二来计算 1300 * 25% 1500 * 22%
c.增长率远超100%时，用公式二来比大小 2000 3000% 2000/0.33 2000 * 300%

出题标志
1、 比较：增量最大/增加最多/增加值最大 对总量增加影响最大(绝对值最大)
2、 计算：增加了/增加/增产/增产量/增加值为… + 量的单位

影响最大、最小 => 取绝对值

其中：国有及国有控股企业 299.8 35.7

A.国有及国有控股企业 B.私营企业 C.股份制企业 D.大中型企业

比重、百分数、百分点

比重 = 部分 / 总量
总量 = 部分 / 比重
部分 = 总量 * 比重
利润率 = 利润 / 营业收入
资产负载率 = 负债额 / 资产总额
人口增长率 = 出生率 - 死亡率

比重：总体中某部分 占 总体的百分比

部分 => 总体 包含关系 比谁减谁，比谁除以谁

百分数：量A占量B的百分比例： (A / B) * 100% 常见考点：增长率

百分点：两个百分数做差的结果 a% - b% = c%

1、 量与量比较，只能用百分数

例：甲占乙的X% 甲 / 乙
甲比乙的X% (甲 - 乙) / 乙
乙比甲的X% (乙 - 甲) / 甲 绝对值
今年比去年增加/减少X% (现期 - 基期) / 基期

2、 率与率比较，一般用百分点
直接做差，后减前(时间上的后和前) 也就是比谁减谁

在同一个总量中，比重可以代替量来计算

比例赋值：把总量赋值为100

例：在2007年我国文化、体育和娱乐劳动力需求人数 比 水利、环境和公共设施管理业 多()
A.75% B.133% C.176% D.198%

(1.4 - 0.6) / 0.6 = 4 / 3

指数 P32

指数：用于衡量某种要素相对变化的指标里
常见的指数包括：纳斯达克指数、物价指数、上证指数、区域价格指数

1、 环比指数(年环比)
a. 指数只反应增长率，不体现具体的量(各年的基期都是100)
b. 指数的基期量，通常都是100(好计算增长率)
c. 指数的单位是点，这个点与百分点没有关系

2、 定基指数
各期指数都以某一固定量作为基期
定基指数的数字直接当做具体量来计算

定基指数2000 2005 2006 2007 2008

(800 - 600) / 800

翻番(番为整数)

翻一番为原来的2倍；
翻两番为原来的4倍；
依此类推，翻 N 番为原来的 2^n 倍

顺差、逆差 P35、P46

顺差(出超) = 出口 - 进口
逆差(入超) = 进口 - 出口

对外贸易顺差：在一个时期内，一个国际(或地区)的出口商品额 大于 进口商品额

对外贸易逆差：在一个时期内，一个国际(或地区)的出口商品额 小于 进口商品额

1、 顺差，出 > 进；逆差，进 > 出
2、 净进口额 = 进口额 - 出口额
净出口额 = 出口额 - 进口额
3、 对外贸易总额 = 进出口总额 = 进口额 + 出口额
4、 对外贸易差额 = |进口 - 出口|
5、 进出口总额增长率一定是混合增长率
6、 主题转换：法 => 中 出口：顺差
中 => 法 进口：逆差

三大产业(三次产业) P67

第一产业(一产业)：农业(种植业、林业、牧业、副业、渔业)
第二产业(二产业)：工业和建筑业
第三产业(三产业)：除第一、二产业以外的其他各业，也就是服务业

三大产业的产值 = 增加值(特殊情况)
GDP = 三大产业的产值

增加值：
1、 可当做增长量来用
2、 也可当做主语

人口自然增长

公式：
人口自然增长量 = 出生人数 - 死亡人数
人口自然增长率 = 人口出生率 - 人口死亡率
单位：
人口数：千人，百万，十亿…(注意：不要默认单位)
人口自然增长率：‰ (资料中通常是‰，问题里可能是‰，注意：单位可能不一样)

计算时看单位

GDP(国内生产总值)

指一个国家(地区)所有常住单位在核算期内生产活动的最终结果

一、二、三产业的增加值之和构成了国内生产总值

GNP(国民生产总值)

指一个国家（或地区）所有国民在一定时期内生产的最终产品和服务价值的总和

CPI(居民消费价格指数)

反映居民家庭购买消费品及服务价格水平变动情况的相对数（指数的基期数值定为一百）

它是进行经济分析和决策、价格总水平监测和调控及国民经济核算的重要指标

其按年度计算的居民消费价格指数变动率通常被用来作为反映通货膨胀或紧缩程度的指标

基尼系数

定量测定收入分配差异程度

国际上通用的，用以衡量一个国家或地区人民收入差距的常用指标

介于0–1之间，0分配趋向于平等；
越大，代表收入差距越大；

恩格尔系数

衡量一个国家或地区人民生活水平的常用指标，比例越低，代表生活水平越高

恩格尔系数是食品支出总额占个人消费支出总额的比重
