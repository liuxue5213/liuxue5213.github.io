第一天

发布 : 2020-09-09 分类 : 公考 浏览 :

一、 自我介绍

大家好，先自我介绍一下，我是面试讲师@@，
首先恭喜大家通过笔试，进入面试环节；
在上岸的道路上，只差最后一步，
接下来几天，我将陪同大家 一起迈过这最后一步；

大家之所以到这里来，都是为了学习面试的，有一个问题 大家有没有想过

1.1 国家为什么要设置面试环节呢 为什么不直接通过笔试选拔呢

通过面试，国家可以选拔到 更符合工作岗位要求的人才，
其次，也是通过面对面的方式 直接考察考生的基本素养、语言表达、个人综合能力，
从而判断考生是否与岗位匹配的流程，不然只考笔试就可以了

1.3 心态

在学习之前 我们大家要先有一个好的心态，这个心态很重要，如果想学到更多的知识，就要先把自己想象成“一个空的杯子”，这样才能学习到更多的知识；

在学习的过程中，大多数考生都会遇到以下几个瓶颈期

思维和思维的碰撞
1-2天 明白思路但不会用，无法开口答题
首先要增加自己的知识储备，名人名言，时事热点的了解；其次多说多答，尝试多写一些题，理解思路

思维和内容的碰撞
1-4天 感觉自己没有进步，不想答题，自我否定
每一道题可以增加知识储备，听其他人答题取长补短，引入金句

内容精进的瓶颈期
4天之后随时出现 答题内容过于冗长，有时会出现跑题的情况
通过录音，反复对比，不断提高答题质量

考场状态的瓶颈期
题量在300-400左右 答题没有问题，但是情感和自如度有影响
使用一语道破思路，观点前置，答题时增加感情投入，要有自己的见解，表达出自己的真情实感

二 课程安排

大家把讲义翻到第一页，课程安排
前4天讲理论 后3天都是刷题和模拟演练

今天我们的安排是
上午讲考情和社会认知类题型
下午社会认知复合类题型
晚上带大家刷题 回顾今天白天所讲的知识

我们来学习面试，
首先要知道面试考什么，考试方式是什么，题型是什么，
看一起看咱们河北省考情介绍

三、 考情介绍

1. 考试形式

结构化面试 半结构化面试 无领导小组讨论 结构化小组讨论 稍后会具体来讲

2. 历年考试时间情况

16年 5月底6月初
17年 5月底6月初
18年 5月中旬6月初
19年 6月底7月初

3. 考试时间及地点

预测20年的考试时间 5-7月之间
地点：学校、宾馆、机关单位

4. 面试题量

从近几年咱们河北省公务员的面试情况来看，基本上都是每天2套题，上下午各一套，每套里面有3道题；
但值得我们大家注意的是，从2016年开始，
首次减少了面试题目的数量，4道题=3道题；
答题时间也随之相应改变，20分钟=》10分钟；
但相较于其他的省份，答题时间是较短，
所以大家在备考的时候 一定要注意控制好时间，这个是十分重要的；
如果答题时 接近超时了 立即收尾总结 ，超时会扣分的，

5. 面试时间

10分钟3道题，包括听题或者看题、思考、回答的总时间

6. 考试方式

6.1 题目

看题 念题 看题和念题

6.2 笔纸

部分（看情况），如果没有，需要通过思考来分析

6.3 考官

考官普遍在5-7人之间，正中央大概率是主考官，
或者谁说话，谁为主考官；
考官两边会有记分员、计时员、监督员和摄像机；

考官的组成：
任职5年以上的在职官员
来自各个单位人事、纪检部门的骨干
副教授职称以上的老师（党校居多） 

主考官：来自组织、人事部门、科研院所等
在职官员：
    国考省考：副处级及以上
    事业编制：副科级及以上

在考试过程中 一定要与每一位考官 眼神交流1句话以上的时间

6.5 钟表

这是几乎所有考生都会问的问题

只允许携带机械手表，电子产品类型的一律不可以携带，
而且之前的考生也反映过，在答题过程中 没有时间去看

1、考生桌子
2、墙上挂表
3、考官桌子上有倒计时屏幕/电脑
4、没有表

不过大家也不用太过于担心时间问题，我们会在后续的课程中，
通过大量的刷题，养成一个快速思考的习惯，其他的机构一个课程下来，也就20-30道题，我们后面会进行至少200道题的锻炼，不断提高大家的答题能力，不要认为这样有很大的压力，这样的好处是 将来在上考场时 可能遇到一道题 就有一种似曾相识的感觉 可以更好的帮助大家上岸

同样刷题是针对每一种题型的，还有我们特有的十分八遍的方法和脱敏训练，更好地帮助大家敢于开口说话；

7. 面试成绩

打分方式：去掉最高分和最低分 计算平均值
出分：
考完试，直接走，成绩网上公示，
考完试，直接走，每半天考场外张贴，
考完试，直接到等候室，等工作人员念分或者张贴成绩
考完试，考场内听完再立场；下一个考完试，上一个回去看分，签字

四、 考试形式

结构化面试、半结构化面试、无领导小组讨论、结构化小组讨论（三人小组）

1. 结构化面试

也叫面谈，每个人面对的考官组成、考题、评分标准、面试流程都是一样的，好处是可以保证考试公平公正

特点：标准化 结构化 优点 规范性 缺点 模式化 互动性不足（考官说的）

反模式化
互动性不足=》表现 动作 眼神 声音 加大交流
内容贴近 真实情感

2. 半结构化面试

在结构化的基础上，加入随机提问(银行、企业)

3. 无领导小组讨论（国考 省考 银行 个别地方法院、检察院、事业单位）

5-12位考生临时分为一组，同时进入考场，大家分别讨论对一个话题的看法，或者制定一个计划，
解决一个问题，提出一个方案，旁边有评分员进行打分

好处是可以考察到笔试不易观察的维度，对实际行为的考察，考生不容易演示，
考察的能力比较广

不足：题目设计难（有争议的题目） 对考官要求大（过程中很难判断考生的品性） 考生会受到其他人的影响（比如自己的观点被否定） 考生仍有装好的可能（吸引他人意见 考官无法识别）

4. 结构化小组讨论（三人小组）

国税在延用，三人一组，进入考场后先进行结构化面试提问，轮流回答并点评

五、 考试题型

怎么看
对社会现象、道理的认知和理解、感受
绝大多数必考，偏向于想象类【时政】、哲理类【习总书记的讲话】、漫画类
谈谈你对佛系青年的认识
互联网发展，网公安机关打击网络谣言，你怎么看

怎么办
日常生活中的人际交往，事务、情况和处理
矛盾处理偏多【人与人】、组织流程【宣传】组织处理较少
领导安排任务，同事不愿配合，你怎么办
组织预防电信诈骗的活动，你怎么办

怎么说
以演讲、讲故事、情景模拟等方式回答问题
演讲题【一般性主题，如创新、团结、坚持】【漫画演讲或者结合时政】、情景模拟较少
以一个词语进行演讲、旁边的邻居因为吵闹有投诉，你如何说服邻居，请现场模拟

你是谁
对个人情况、求职动机的表述
自己对岗位的看法、优势特长

上下午题型都是类似的，2019年漫画题突出

2019 国家发展阶段、领导干部用权 人才引进 老城改造
2018 名人名言、真理类、习总书记讲话、工作效率、人际处理、短视频、网络问题、终身学习、传统文化、互联网、乡村振兴、扶贫问题、群众沟通、青年、廉政、大学生就业、政策落实、基层工作
2017 老年人保健品骗局、标题党、减政便民、遛娃经济、共享单车、脱贫攻坚、自主招生、高离婚率、出轨率、限塑令

考试流程

国家为什么要设置面试环节呢 为什么不直接通过笔试选拔呢

通过面试，国家可以选拔到 更符合工作岗位要求的人才，
其次，也是通过面对面的方式 直接考察考生的基本素养、语言表达、个人综合能力，
从而判断考生是否与岗位匹配的流程，不然只考笔试就可以了

如何看待面试答题内容和答题形式

答题内容 针对题目回答的内容
答题形式 答题的精气神、重音、断句 —脱敏训练：学员拉到户外进行练习，来达到克服心理紧张的目的

想给考官留下怎样的印象

考官在看什么

语言表达能力
表达准确 简洁 流畅得体 无语病

综合分析能力
层次清晰 主次分明 条理清晰 善于综合分析 逻辑性强 思维面广（答题时 要多个角度去论述）

求职动机
诚实负责，自信而又务实，既有进取意识而又踏实、谦逊、回答极有分寸

计划组织能力
有极强的设计能力，计划能力，计划周密可行，组织能力极强，有极强的合作意识，合作技巧高，协调沟通方法得当
协调应变能力

举止仪表
文化素养高，举止文雅，穿着得体，无多余动作（
一些小动作，转笔、抖腿等会扣分
）

面试特点-逆面试

逆 不一样，和其他同学的答题思路、方式不一样、有自己的特点

一逆 逆授课风格
气势 答 讲 听
二逆 逆思路
答题思路 大众答法 短时间开口；一语道破 一个核心内容；
三逆 逆内容
仰望星空（背诵资料、语句） 脚踏实地（挖掘发现身边的故事，背诵例子 没有感情 背的东西都是一样的 自己的东西才是最好的）
四逆 逆训练方式
性格测试 决定以后答题的瓶颈
脱敏训练-脸皮厚
十分八遍-开口难 不自信 对内容过于依赖的 不一定要将所有点都想明白 面试的时候 没有时间弄明白
第一遍 一句话 第二遍 一点一点增加 说的话就是有理 每个人理解的观念不同 总会有表达不全的 不要太过于纠结
五逆 逆状态
表演式答题 真诚
背答案 自己会的没说 不会的说不好

尽量说自己说的好的 让别人听懂
而不是背诵的高大上 别人听不明白

确定面试对象
攻擂 岗位招1个 笔试前3名进面 自己成绩在2-3
守擂 岗位招2个 笔试前6名进面 自己成绩在1-2
资格审核
报名信息是否真实，如果报名信息和提供的材料不符合，被淘汰，不能参加，根据笔试排名补录
面审、网上邮寄
面试报道
提前准备好服装、证件、材料

抽签

考生序号签 在考场抽完之后
