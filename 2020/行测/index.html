行测-数量关系

发布 : 2020-10-10 分类 : 公考 浏览 :

数学运算主要考察 考生理解、把握事物间量化关系和解决数量关系问题的能力。

主要涉及数据关系的分析、理解、判断、运算等。

常见的题型：数学运算

一、数学规律

1.1 数字特性

数字特性思想是指不直接求得最终结果，儿只需要考虑最终计算结果的某种“数字特性”，从而达到排除错误选项的方法。

掌握数字特性思想的关键，是掌握一些最基本的数字特性规律（仅限自然数内讨论）

奇偶性

基础
奇数 + 奇数 = 偶数
偶数 + 偶数 = 偶数
奇数 + 偶数 = 奇数

奇数 * 奇数 = 奇数
奇数 * 偶数 = 偶数
偶数 * 偶数 = 偶数

推论
两数的和与差 奇偶性相同，求和找差，求差找和
若干数相乘，一偶则偶，全奇才奇

总数 * (1 / N) = 整数  => 总数为N的倍数
总数 = 部分 * (N / M) 或 (部分 / 总数) = (N / M) => 部分为N的倍数 总数为M的倍数

多位数问题，优先考虑3或9的倍数

倍数关系

常用的倍数 2、3、5、9  11、25
2的倍数：个位数是2/4/6/8/0
3的倍数：所有数字之和可以被3整除
5的倍数：个位数是0/5
9的倍数：所有数字之和可以被9整除 注意：9的倍数比3的倍数范围更小

11的倍数：奇数项的数字和 - 偶数项的数字和 = |0/11|
25的倍数：末两位是25/50/75/00

注意：2或5整除看末位 4或25整除看末两位 8或125整除看末3位

如果 a:b = m:n (m和n是最简整数比，也就是互质，没有公约数)
则以下结论成立
    a是m的倍数，b是n的倍数
    a+b 是 m+n 的倍数，a-b 是 m-n 的倍数
推论：
    a:b:c... = m:n:p...  则 a+b+c... 是 m+n+p... 的倍数
总结：
    出现比例、分数、百分数、倍数、余数问题，优先考虑倍数关系；
    a,b 均为整数时(人数、件数等)，上述结论一定成立；
    a,b 不一定为整数时，上述结论大概率成立

三、行程问题

核心公式：路程 = 速度 * 时间   S = V * T

1. 平均速度
S一定，平均速度 = 2V1V2 / （V1 + V2）
T一定，平均速度 = (V1 + V2) / 2
结论：路程一定，等时间的平均速度 比 等距离的平均速度 快

2.相遇追及
相遇：相向而行，两者共同走完S 公式：(V1 + V2) * t = S
     多次相遇：第N次相遇，两者走的总路程 = (2N - 1) * 单程
追及：同向而行，快者 比 慢者 多走S  公式：(V1 - V2) * t = S     

3. 比例路程
路程一定，速度和时间成反比
时间一定，速度和路程成正比
速度一定，时间和路程成正比

4. 环形跑道(从同一点出发，一圈的路程为S)
相遇：反向，相遇 n 次两者共同走完 n 圈。 公式： (V1 + V2) * t = n * S
相向：同向，快者超过/追上慢者 n 次，就比他多走 n 圈。  公式： (V1 - V2) * t = n * S

5. 流水行船

V顺 = V船 + V水
V逆 = V船 - V水
V船 = (V顺 + V逆) / 2
V水 = (V顺 - V逆) / 2

用数列的方法去构造最多和最少

c.抽屉原理/鸽巣原理

标志：至少...同一...

4.3 植树问题

七、经济利益问题

1. 总价 = 单价 * 销售量
2. 利润 = 售价 - 成本
3. 总利润 = 单件利润 * 销售量
4. 利润率 = 利润 / 成本 = (售价 - 成本) / 成本 = (售价 / 成本) - 1
5. 定价 = 成本 * (1 + 期望利润率)
6. 几折：既 现价 为 原价 的百分之几十

单价 = 定价 = 售价
成本 = 进价

资料分析中 利润率 = 利润 / 营业收入

八、溶液问题

溶液的组成成份和分类
    1. 溶质：被溶解的物质
    2. 溶剂：能溶解其他物质的物质    通常使用水(有水参加的溶剂)
    3. 饱和溶液：溶质不能继续被溶解的溶液
    4. 不饱和溶液：溶质可以继续被溶解的溶液

溶液问题的基本公式
    溶液 = 溶质 + 溶剂
    溶质 = 溶液 * 浓度 = (溶质 + 溶剂) * 浓度
    浓度 = 溶质 / 溶液
    混合浓度 = 总溶质 / 总溶液

a. 溶液稀释问题(看谁没有变)
    核心思路：找定量

b. 混合问题
    题型识别：两种溶液混合(三种)

    混合问题的核心思路
        混合后的浓度、增长率、平均数的值居中(居中不正中，偏向量大的)
        混合前谁的量大，谁对于混合后的贡献最大(倾向贡献大的一方)
        混合浓度 = 总溶质 / 总溶液

9.3 下次相遇问题

求各周期最小公倍数，算出新的周期
注意：每隔 n 天 = 每 n+1 天，每 n+1 天 = 周期是 n+1 天

结论：计算下次相遇，统一去头

十、杂项问题

10.1 过河问题

M个人过河，船能载N个人，需要A个人划船(默认1)，相当于每次能载过去 N-A 个人；
共需过河 M-A / N-A (有余数必进1)

计算的结果为趟，往返为1趟

10.2 青蛙爬井

井的深度是M，青蛙每天爬N，每次滑下来A，相当于每天能爬 N-A
青蛙从井里跳出来需要 M-A / N-A (有余数必进1)

10.3 空瓶换酒

M个空瓶换1瓶酒，相当于 M-1 个瓶子可以喝一瓶酒(不带瓶)；
如果有A个空瓶，M个空瓶可以换1瓶酒，可以换的酒数量是 A / M-1 (取整数)

10.4 烙饼问题

需要烙饼的张数是A，面数是2A，锅里每次能放M张，没面烙的时间是T；
烙饼的总时间 (2A / M) * T （注意：括号里的算式有余数必进1）

10.5 天秤问题

3的N次方个小球中，有1个特殊(轻)，需要用天平秤 N 次(就高不就低)；

操作型题目，问最小一般选最小(80%以上) 1次不科学

数字推理

大胆尝试，小心求证

一、 多级数列

做商特点：变化快， 整整整分，可拆分–倍数关系

做和特点：大小变化不规则

因式分解：项数少；可拆分–倍数关系
找关系(-2，-1，0，1，2，3) （1，3，5，7，9）

满足三项以上的特性才可以当做规律，递增平缓，推荐做差

二、 多重数列

长数列要分组：长数列指的是 项数>= 8

数列中有两个括号的：先交叉，再分组

分组的种类：两两分组，先加减乘除，再考虑平方；
三三分组，两个数求和或做差 等于 第三个数

三、 分数数列

1、 跳过被约分项

2、 利用未被约分项推导

四、 幂次数列

1、幂次方

2、负幂次方

3、幂次方修正

五、 多重数列

观察数列的整体变化趋势

1、 整体递减：做差或者做商

2、 整体递增：增长较缓–做和，增长较快–乘积，增长急速–平方
