虚拟机

发布 : 2017-01-15 分类 : 笔记 浏览 :

一、下载、安装准备

1.1 下载VMware

1.2 下载虚拟机镜像文件

1.3 打开安装镜像

首先右键 以管理员身份运行VM(或者给安装目录权限 右键VM安装文件夹目录->属性->安全下面设置)
打开 选择下载的镜像文件，设置虚拟机名称和存储目录
导入完成后，编辑->虚拟网络编辑器 将VMnet设置为桥接，桥接到计算机上网的网卡上面

1.4 配置共享文件夹目录

虚拟机设置->选项->共享文件夹->配置总是启用、添加本地项目文件夹
