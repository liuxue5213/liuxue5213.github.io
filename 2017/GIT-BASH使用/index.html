GIT-BASH使用

发布 : 2017-03-24 分类 : LINUX 浏览 :

创建ssh

检查是否已经存在ssh 在local打开terminal：$cd ～/.ssh 如果存在，先将已有的ssh备份，或者将新建的ssh生成到另外的目录下
如果不存在，通过默认的参数直接生成ssh 生成过程如下：

在github中添加ssh

登陆github，选择Account Settings–>SSH Public Keys 添加ssh Key：打开你生成的id_rsa.pub文件，将其中内容拷贝
测试SSH：

git config –global user.name “your real name”
git config –global user.email “xxxxx@gmail.com“

git init
git commit -a

git remote add navgeek git@github.com:liuxue5213/navgeek.git

git push navgeek master

github.com:liuxue5213/navgeek.git

git pull git@github.com:liuxue5213/navgeek.git

git stash
do some work
git stash pop

git stash apply stash@{1}

git stash list

save uncommitted changes

git stash

pull, edit, etc.

list stashed changes in this git

git stash list

see the last stash

git show stash@{0}

apply last stash and remove it from the list

git stash pop

for more info

git stash –help
