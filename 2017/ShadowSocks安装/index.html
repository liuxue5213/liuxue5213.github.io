shadowsocks安装

发布 : 2017-10-28 分类 : 笔记 浏览 :

一、shadowsocks安装

1.首先安装WGET

2.下载shadowsocks

3.修改文件权限、进行安装

目前使用的python版本 然后输入密码、默认端口号、加密方式 最后开始安装

3.根据成功安装提示 使用shadowsocks进行连接

二、BSP多用户、流量限制管理工具

1.先修改文件权限、并对端口和密码设置调整

2.修改完之后、重启shadowsocks

3.下载、安装流量限制管理工具BSP

bsp -p 8989 -P 123456 -s 100 -a -A -j

bsp –start

bsp -p 8989 -c
