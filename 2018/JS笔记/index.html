HTML
块级元素 行内元素内联元素
块级元素独占一行 可以设置width height 默认100% div
内联元素无效 根据自身内容或者子元素决定 span a img
display block块元素 inline 行内元素
inline-block 元素在行内显示 又能设置宽高
块元素 从上到下排列 遇到内联样式 从左到右 普通流

margin border padding content
总宽度=margin-left+border-left+padding-left+width+padding-right-border-right+margin-right
width:200px padding:10px border:5px soild #000 margin:20px
200+102+52+202=270px
小于IE9 200+202=240px

posittion 默认值static 正常位置 没有定位 无法使用top right等移动
relative 相对定位 相对于元素的正常位置来进行定位 可以使用top right
absolute 绝对定位 元素在页面不占据为知 相对于最近一级的定位不是static的父元素进行定位
fixed 绝对定位 相对于浏览器窗口来进行定位 和absolute一样 特殊的absolute
inherit 从父元素继承position属性的值

try{t2=arg2.toString().split(“.”)[1].length}catch(e){}
